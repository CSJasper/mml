## 해석기하학

**해석기하학(Analytic Geometry)** 이란 여러 개의 수로 이뤄진 순서쌍을 기하학적으로 나타내는 방법을 부르는 이름이다.

좌표기하학 또는 카테시안 기하학이라고도 불리우며, n개의 순서쌍의 수를 미지수로 하는 방정식의 형태로 도형의 성질을 설명한다. 

여러 머신 러닝 알고리즘들, 특히 딥러닝 알고리즘들을 이해하고 사용하려면 해석기하학을 잘 알아야할 필요가 있다.
