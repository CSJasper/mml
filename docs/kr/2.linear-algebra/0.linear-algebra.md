# Linear Algebra
{: .no_toc }


Linear Algebra Introduction
{: .fs-5 .fw-300 }

---


선형대수(Linear Algebra)는 벡터와 벡터공간, 벡터를 다루는 방법들을 연구하는 대수학의 한 분야이다.

현대 선형대수학은 그중에서도 벡터 공간이 주 연구 대상이다. 

보통 사람들이 학교에서 배우고, 일반적으로 알고있는 벡터는 기하벡터(geometric vector)라고 불리우며,

$$\vec { x }, \vec { y }$$
